Как это работает:

- @echo off — подавляет вывод команд в консоль.
- echo.| clip — посылает пустую строку в буфер обмена, тем самым очищая его.
- exit — завершает выполнение bat-файла.

Как использовать:

1. Запустите bat-файл двойным кликом или через командную строку.

Буфер обмена будет очищен моментально!
